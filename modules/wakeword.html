Wakeword Module

class modules.wakeword.vosk.Wakeword(model_path=None, prober_instance=None, shared_state=None)

Bases: object

Represents the wakeword detector, it work with the Vosk library and the Levenshtein distance algorithm in due to get a better performance.

Attributes:

model (object): An instance of the Vosk model class. rec (object): An instance of the KaldiRecognizer class. stream (object): An instance of the RawInputStream class. q (object): An instance of the Queue class. wakeword (str): The wakeword to be detected. callback (function): The callback function to be called when the wakeword is detected.
