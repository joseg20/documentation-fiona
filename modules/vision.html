class modules.vision.vision.Vision(prober_instance=None, camera_index=0)

Represents the Vision module of Fiona, responsible for processing video frames and coordinating face and hand detection functionalities.

class modules.vision.face.Face(prober_instance=None, frame=None, deadspace=0.35)

Bases: object

Represents the Face detection and processing functionality within the Vision module of Fiona.

This class is responsible for detecting faces in video frames, drawing debug lines and face rectangles, and managing face data.

Attributes:

prober_instance (object): An instance of the prober module, used to register events. deadspace (float): Margin around the detected face area where no processing is done. saved_faces_path (Path): Path to the saved faces data file. images_path (Path): Path to the directory containing images. saved_faces (dict): Dictionary containing encodings and names of saved faces. face_list_cords (List): List of coordinates of detected faces. frame (np.ndarray): Current frame for face detection.

class modules.vision.hand.Hand(prober_instance=None, detectionCon=0.85, maxHands=1)

Bases: object

Represents the Hand detection and processing functionality within the Vision module of Fiona.

This class is responsible for detecting hands in video frames and drawing the count of detected fingers.

Attributes:

prober_instance (object): An instance of the prober module, used to register events. detectionCon (float): Detection confidence threshold for the hand detector. maxHands (int): Maximum number of hands to detect. detector (HandDetector): Hand detector object from cvzone. finger_count (int): Count of fingers detected in the current frame. face_list_cords (list): List of coordinates of detected faces (currently unused). font (cv2.FONT_*): Font used for drawing text on frames. font_scale (float): Scale of the font used in drawing. font_thikness (int): Thickness of the font used in drawing. font_color (tuple): Color of the font used in drawing. background_color (tuple): Background color used in drawing.
